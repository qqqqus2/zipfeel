# Zipfeel Storybook

Next.js + shadcn/ui + Tailwind CSS + Storybook 프로젝트에 오신 것을 환영합니다!

## 프로젝트 구성

이 프로젝트는 다음 기술들로 구성되어 있습니다:

- **Next.js 16** - React 프레임워크
- **TypeScript** - 타입 안정성
- **Tailwind CSS v4** - 유틸리티 우선 CSS 프레임워크
- **shadcn/ui** - 재사용 가능한 컴포넌트 라이브러리
- **Storybook 8** - UI 컴포넌트 개발 및 문서화 도구

## 컴포넌트 추가하기

shadcn/ui 컴포넌트를 추가하려면:

```bash
npx shadcn@latest add [component-name]
```

예시:
- `npx shadcn@latest add card`
- `npx shadcn@latest add input`
- `npx shadcn@latest add dialog`

## Story 작성하기

각 컴포넌트에 대한 Story를 작성하여 다양한 상태와 variant를 시각화할 수 있습니다.

Story 파일은 `*.stories.tsx` 형식으로 작성하면 됩니다.
